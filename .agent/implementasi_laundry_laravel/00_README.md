# Panduan Implementasi Sistem Booking dan Monitoring Laundry

Judul project:

**Implementasi Sistem Booking dan Monitoring Laundry Berbasis Web Menggunakan Laravel**

Stack utama:

- Laravel
- Blade Template
- TailwindCSS
- PostgreSQL
- Eloquent ORM
- Laravel Sail
- Docker
- React Component via Vite
- shadcn/ui / Radix UI / Framer Motion / Lenis sebagai enhancement UI

Urutan pengerjaan:

1. Setup Project
2. Setup Database PostgreSQL
3. Auth dan Role User
4. Master Data Layanan Laundry
5. Booking Laundry
6. Monitoring Status Cucian
7. Transaksi dan Pembayaran
8. Cetak Nota
9. Dashboard Statistik
10. Laporan
11. React Component Enhancement

Target alur demo:

User booking laundry → kasir menerima → status cucian berubah → transaksi pembayaran → cetak nota → owner melihat laporan.
